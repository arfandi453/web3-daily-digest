## 🔍 What is web3-daily-digest?

web3-daily-digest collects news from more than 30 top cryptocurrency and blockchain sources. It delivers a daily summary of important stories from websites like CoinDesk, Decrypt, and key figures such as Vitalik Buterin and Arthur Hayes.  

The app runs on Windows without needing extra software or API keys. It shows important web3 and crypto updates in one place.  

You do not need any programming knowledge to use it.

## 🛠 Troubleshooting Tips

**If the app won’t start:**  
- Check if Windows Defender or antivirus software blocked the installer. Whitelist the app or allow permissions.  
- Make sure your Windows version is supported (Windows 10 or newer).  
- Restart your computer and try again.

**If news does not load:**  
- Verify your internet connection is active.  
- If you use a VPN or firewall, ensure they allow the app’s access.  
- Try closing and reopening the app.

**If downloads fail:**  
- Download the file from the releases page again to avoid incomplete files.  
- Use a stable internet connection for the download to complete.
